# 📱 Настройка приложения для проверки в App Store Connect

## ❌ Что нужно исправить

Для отправки приложения на проверку необходимо заполнить:

1. ✅ Снимок экрана для iPad Pro (13-дюймовый дисплей)
2. ✅ Снимок экрана для iPhone (6,5-дюймовый дисплей)
3. ✅ Основная категория приложения
4. ✅ Ценовой ярус в разделе Ценообразование
5. ✅ URL-адрес политики конфиденциальности

---

## 📋 Шаг 1: Снимки экрана

### 1.1. Снимок для iPhone (6,5-дюймовый дисплей)

**Требования:**
- Размер: 1284 x 2778 пикселей (iPhone 14 Pro Max, iPhone 13 Pro Max, iPhone 12 Pro Max)
- Формат: PNG или JPEG
- Минимум: 1 снимок, максимум: 10 снимков

**Как сделать:**
1. Запустите приложение на симуляторе iPhone 14 Pro Max (или реальном устройстве)
2. Сделайте скриншот (⌘+S на Mac или через инструменты разработчика)
3. Или используйте готовые скриншоты из проекта

**Где загрузить:**
- App Store Connect → Ваше приложение → App Store → Версия для iOS
- Раздел "Снимки экрана" → iPhone 6,5"

### 1.2. Снимок для iPad Pro (13-дюймовый дисплей)

**Требования:**
- Размер: 2048 x 2732 пикселей (iPad Pro 12.9")
- Формат: PNG или JPEG
- Минимум: 1 снимок, максимум: 10 снимков

**Как сделать:**
1. Запустите приложение на симуляторе iPad Pro 12.9"
2. Сделайте скриншот
3. Или используйте готовые скриншоты из проекта

**Где загрузить:**
- App Store Connect → Ваше приложение → App Store → Версия для iOS
- Раздел "Снимки экрана" → iPad Pro (12.9")

**Примечание:** Если приложение только для iPhone, можно загрузить те же скриншоты, но в размере для iPad.

---

## 📋 Шаг 2: Основная категория приложения

**Где указать:**
- App Store Connect → Ваше приложение → App Store → Информация о приложении
- Раздел "Категории"

**Выберите основную категорию:**
- Для SPA приложения подойдут:
  - **"Здоровье и фитнес"** (Health & Fitness)
  - **"Образ жизни"** (Lifestyle)
  - **"Медицина"** (Medical) - если есть медицинские услуги
  - **"Красота"** (Beauty) - если акцент на красоте

**Рекомендация:** Выберите **"Здоровье и фитнес"** или **"Образ жизни"**

---

## 📋 Шаг 3: Ценовой ярус

**Где указать:**
- App Store Connect → Ваше приложение → App Store → Ценообразование и доступность
- Раздел "Ценообразование"

**Выберите ценовой ярус:**
- **Бесплатно** (Free) - если приложение бесплатное
- **0,99 USD** - минимальная платная цена
- Или выберите другую цену

**Рекомендация:** Если приложение бесплатное, выберите **"Бесплатно"**

---

## 📋 Шаг 4: Политика конфиденциальности

### 4.1. Создайте политику конфиденциальности

**Что должно быть в политике:**
- Какие данные собирает приложение
- Как используются данные
- Кто имеет доступ к данным
- Как защищаются данные
- Контактная информация

**Пример структуры:**
```
1. Введение
2. Какие данные мы собираем
3. Как мы используем данные
4. Передача данных третьим лицам
5. Безопасность данных
6. Ваши права
7. Изменения в политике
8. Контакты
```

### 4.2. Разместите политику в интернете

**Варианты:**
1. **На вашем сайте** (если есть):
   - Создайте страницу: `https://ваш-сайт.com/privacy-policy`
   - Разместите текст политики

2. **Используйте генератор политик:**
   - [Privacy Policy Generator](https://www.privacypolicygenerator.info/)
   - [Termly](https://termly.io/)
   - [FreePrivacyPolicy](https://www.freeprivacypolicy.com/)

3. **GitHub Pages** (бесплатно):
   - Создайте репозиторий на GitHub
   - Включите GitHub Pages
   - Разместите HTML страницу с политикой
   - URL будет: `https://ваш-username.github.io/privacy-policy`

### 4.3. Укажите URL в App Store Connect

**Где указать:**
- App Store Connect → Ваше приложение → App Store → Конфиденциальность приложения
- Раздел "Политика конфиденциальности"
- Вставьте URL: `https://ваш-сайт.com/privacy-policy`

## 📋 Шаг 5: Дополнительная информация (если нужно)

### 5.1. Описание приложения

**Где указать:**
- App Store Connect → Ваше приложение → App Store → Версия для iOS
- Раздел "Описание"

**Что написать:**
- Краткое описание приложения (до 4000 символов)
- Основные функции
- Для кого предназначено

### 5.2. Ключевые слова

**Где указать:**
- App Store Connect → Ваше приложение → App Store → Версия для iOS
- Раздел "Ключевые слова"

**Что указать:**
- До 100 символов
- Через запятую, без пробелов
- Например: `spa,здоровье,релакс,массаж,красота`

### 5.3. Иконка приложения

**Требования:**
- Размер: 1024 x 1024 пикселей
- Формат: PNG или JPEG
- Без прозрачности
- Без закругленных углов (Apple добавит автоматически)

**Где загрузить:**
- App Store Connect → Ваше приложение → App Store → Версия для iOS
- Раздел "Иконка приложения"

## 🚀 После заполнения всех полей

1. Проверьте все разделы еще раз
2. Нажмите **"Отправить на проверку"** или **"Submit for Review"**
3. Дождитесь проверки (обычно 1-3 дня)
4. После одобрения приложение появится в App Store

## 🆘 Если что-то не работает

### Ошибка: "Снимок экрана не соответствует требованиям"

**Решение:**
- Проверьте размеры (должны быть точными)
- Используйте PNG формат
- Убедитесь, что снимок не содержит прозрачности

### Ошибка: "URL политики конфиденциальности недоступен"

**Решение:**
- Проверьте, что URL работает и доступен публично
- Убедитесь, что страница содержит текст политики
- Проверьте, что нет редиректов

### Ошибка: "Категория не выбрана"

**Решение:**
- Выберите основную категорию (обязательно)
- Вторичную категорию можно не выбирать
