# 🎨 Гайд: Иконки и графические ресурсы для Google Play

## 📋 Требования Google Play

### Обязательные ресурсы:

1. **Иконка приложения (App Icon)**
   - Размер: **512x512 px**
   - Формат: **PNG**
   - Без прозрачности
   - Без закругленных углов (Google Play сам добавит)

2. **Feature Graphic (Промо-баннер)**
   - Размер: **1024x500 px**
   - Формат: **PNG или JPG**
   - Отображается вверху страницы приложения

3. **Скриншоты**
   - Минимум: **2 штуки**
   - Максимум: **8 штук**
   - Рекомендуется: **4-6 штук**
   - Размер: минимум **320px** по меньшей стороне
   - Формат: **PNG или JPG**

## 🖼️ Иконка приложения (512x512 px)

### Текущее расположение:
- Исходный файл: `spa/assets/images/app_icon.png`
- Должен быть размером **1024x1024 px** (для генерации других размеров)

### Требования к иконке:

✅ **Должно быть:**
- Четкое, узнаваемое изображение
- Хорошо видимое на светлом и темном фоне
- Без текста (или минимальный текст)
- Центрированное изображение
- Высокое качество

❌ **Не должно быть:**
- Прозрачный фон (для Google Play)
- Закругленные углы (Google Play сам добавит)
- Мелкие детали, которые не видны в маленьком размере
- Текст, который трудно прочитать

### Подготовка иконки:

1. **Если у вас есть `app_icon.png` (1024x1024):**
   ```bash
   # Используйте его напрямую - он уже правильного размера
   # Просто убедитесь, что он без прозрачности
   ```

2. **Если нужно создать из SVG:**
   ```bash
   # Используйте Inkscape, Illustrator или онлайн-конвертер
   # Откройте logo_priroda.svg
   # Экспортируйте как PNG 1024x1024 px
   # Убедитесь, что фон белый (не прозрачный)
   ```

3. **Создание 512x512 для Google Play:**
   ```bash
   # Используйте ImageMagick, GIMP или онлайн-редактор
   # Откройте app_icon.png (1024x1024)
   # Измените размер до 512x512 px
   # Сохраните как app_icon_512.png
   ```


Это создаст все нужные размеры для Android автоматически.

## 🎨 Feature Graphic (1024x500 px)

### Что это:
Промо-баннер, который отображается вверху страницы приложения в Google Play.

### Требования:
- Размер: **1024x500 px**
- Формат: **PNG или JPG**
- Без важной информации в краях (могут обрезаться на разных устройствах)
- Безопасная зона: оставьте **40px** отступы со всех сторон

### Содержание:
- Название приложения: **Priroda Spa**
- Краткое описание или слоган
- Визуальные элементы (логотип, изображения услуг)
- Призыв к действию (опционально)

### Пример структуры:
```
┌─────────────────────────────────────┐
│  [40px отступ]                      │
│  ┌─────────────────────────────┐   │
│  │   Логотип Priroda Spa       │   │
│  │   "Забота о себе"           │   │
│  │   [Изображения услуг]        │   │
│  └─────────────────────────────┘   │
│  [40px отступ]                      │
└─────────────────────────────────────┘
```

### Инструменты для создания:
- **Figma** (бесплатно, онлайн)
- **Canva** (бесплатно, шаблоны)
- **Photoshop** / **GIMP**
- **Sketch**

## 📸 Скриншоты приложения

### Требования:
- **Минимум:** 2 скриншота
- **Максимум:** 8 скриншотов
- **Рекомендуется:** 4-6 скриншотов
- **Размер:** минимум 320px по меньшей стороне
- **Формат:** PNG или JPG
- **Соотношение сторон:** 16:9 или 9:16 (зависит от ориентации)

### Рекомендуемые экраны для скриншотов:

1. **Главный экран** — список услуг
2. **Экран записи** — выбор мастера и времени
3. **Программа лояльности** — уровни и бонусы
4. **Профиль пользователя** — история визитов
5. **Детали записи** — информация о предстоящем визите
6. **Экран с бонусами** — баланс и история

### Как сделать скриншоты:

#### На Android устройстве:
1. Откройте приложение
2. Перейдите на нужный экран
3. Нажмите кнопки **Power + Volume Down** одновременно
4. Скриншот сохранится в галерее

#### Через эмулятор Android:
1. Запустите эмулятор
2. Откройте приложение
3. В Android Studio: **View → Tool Windows → Device File Explorer**
4. Или используйте встроенный инструмент скриншотов

#### Через Flutter:
```bash
# Запустите приложение
flutter run

# В другом терминале сделайте скриншот
flutter screenshot screenshot_1.png
```

### Обработка скриншотов:

1. **Обрезка:** убедитесь, что скриншоты обрезаны правильно
2. **Обрамление:** можно добавить рамку устройства (опционально)
3. **Аннотации:** можно добавить текст с описанием функций
4. **Единый стиль:** все скриншоты должны быть в одном стиле

### Инструменты для обработки:
- **Figma** — для добавления аннотаций
- **Photoshop** / **GIMP** — для обработки
- **Screenshot Framer** — для добавления рамок устройств
- **Canva** — для быстрого редактирования

## 🛠️ Автоматическая генерация иконок

### Вариант 1: Flutter Launcher Icons (уже настроен)

```bash
cd spa
flutter pub get
flutter pub run flutter_launcher_icons
```

Это создаст все нужные размеры автоматически из `assets/images/app_icon.png`.

### Вариант 2: Скрипт для генерации всех размеров

Создайте файл `scripts/generate_play_icons.py`:

```python
#!/usr/bin/env python3
"""
Генерация иконок для Google Play из исходного изображения
Требуется: pip install Pillow
"""

from PIL import Image
import sys
from pathlib import Path

def generate_icons(source_path: str, output_dir: str = "google_play_icons"):
    """Генерирует все нужные размеры иконок"""
    
    # Размеры для Google Play
    sizes = {
        "app_icon_512": 512,  # Для Google Play
        "app_icon_1024": 1024,  # Исходный размер
    }
    
    source = Path(source_path)
    if not source.exists():
        print(f"❌ Файл не найден: {source_path}")
        return
    
    output = Path(output_dir)
    output.mkdir(exist_ok=True)
    
    img = Image.open(source)
    
    # Конвертируем в RGB если нужно (убираем прозрачность)
    if img.mode in ('RGBA', 'LA', 'P'):
        background = Image.new('RGB', img.size, (255, 255, 255))
        if img.mode == 'P':
            img = img.convert('RGBA')
        background.paste(img, mask=img.split()[-1] if img.mode == 'RGBA' else None)
        img = background
    
    for name, size in sizes.items():
        resized = img.resize((size, size), Image.Resampling.LANCZOS)
        output_path = output / f"{name}.png"
        resized.save(output_path, "PNG", optimize=True)
        print(f"✅ Создан: {output_path} ({size}x{size})")
    
    print(f"\n✅ Все иконки созданы в папке: {output_dir}")

if __name__ == "__main__":
    source = sys.argv[1] if len(sys.argv) > 1 else "spa/assets/images/app_icon.png"
    generate_icons(source)
```

Использование:
```bash
python scripts/generate_play_icons.py spa/assets/images/app_icon.png
```
