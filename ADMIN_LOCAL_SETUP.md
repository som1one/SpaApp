# Локальный запуск админки с бэкендом

## Проблема
При попытке входа в админку показывается ошибка "Неверный email или пароль", потому что:
- Админка подключается к продакшн серверу `http://194.87.187.146:9003`
- Учетные данные для продакшн сервера неизвестны

## Решение: Запуск локального бэкенда

### 1. Запустить базу данных PostgreSQL

```bash
cd backend
docker-compose -f docker-compose.db.yml up -d
```

### 2. Применить миграции

```bash
cd backend
# Установить зависимости (если еще не установлены)
pip install -r requirements.txt

# Применить миграции
alembic upgrade head
```

### 3. Создать супер-админа

```bash
cd backend
python scripts/create_super_admin.py
```

Скрипт создаст админа с данными из `.env`:
- Email: `admin@test.com`
- Password: `admin123`

### 4. Запустить бэкенд

```bash
cd backend
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 5. Открыть админку

Админка уже настроена на локальный бэкенд (см. `admin/.env`):
- URL админки: http://localhost:3001 (или 3002)
- URL бэкенда: http://localhost:8000

### 6. Войти в админку

Используй данные:
- Email: `admin@test.com`
- Password: `admin123`

## Альтернатива: Использовать продакшн сервер

Если нужно подключиться к продакшн серверу:

1. Удали файл `admin/.env` или измени его:
```env
VITE_API_BASE_URL=http://194.87.187.146:9003/api/v1
```

2. Перезапусти админку:
```bash
cd admin
npm run dev
```

3. Узнай учетные данные у владельца продакшн сервера
